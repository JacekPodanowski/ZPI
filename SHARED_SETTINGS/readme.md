# SHARED_SETTINGS

Źródłowe pliki konfiguracyjne współdzielone między Frontend i Backend.

## Użycie

1. Edytuj pliki tutaj (`sizeLimits.json`, `teamRoles.json`)
2. Uruchom update komendą: 
==================================
    .\SHARED_SETTINGS\update.bat
==================================

Pliki zostaną skopiowane do `FRONTEND/src/shared/` i `BACKEND/api/shared/`.
